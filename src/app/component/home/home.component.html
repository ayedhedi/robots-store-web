As the application is deployed on a free Heroku account, when it receives no traffic in a 30 minute period, the web dyno will sleep. In addition to the web dyno sleeping, the worker dyno (if present) will also sleep. When the application is sleeping and receives a web traffic, it will become active again after a short delay. So please wait before refresh the first time you get access to the application.

This WEB application is a POC that has been developed in order to answer to an RBC programming assignment related to the recruitment procedure. Data used in this application is totally generated randomly using a JSON generator
